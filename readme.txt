Демонстрация скрипта для анимации DOM-элементов

1. По кнопке switch to происходит переключение между анимацией с использованием octoAnimator и jQuery
2. Различия в производительности проявляются примерно на 700-1000 элементов (зависит от мощности компа)
3. Количество элеметов задается в поле num (активируется при уходе фокуса с поля)
